# 足球数据分析

利用数学/机器学习/数据挖掘技术来理解足球。希望数据分析能够在未来帮助到中国足球。

### 数据来源

* [Kaggle欧洲足球数据库](https://www.kaggle.com/hugomathien/soccer)
* [Squawka爬虫（因为数据网站原因，此爬虫已不可用）](https://github.com/ucdcoc/squawka-scraper)
